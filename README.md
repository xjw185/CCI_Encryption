CCI: 受约束的混沌迭代加密系统

Constrained Chaotic Iteration Encryption System

基于混沌动力学与几何投影耦合的对称加密原语 | 抗量子计算攻击 | 已开源

📌 一句话介绍

把数据点扔进一个由口令驱动的“混沌搅拌机”里，每搅一次就强行按回一个椭球面上。反复1000次后，记录下它最后的位置作为密文。

🧠 核心原理

```
明文点 + 口令 m
       ↓
   [哈希 → 6个参数]
   (α,β,γ 控制混沌撕扯力度)
   (a,b,c 控制椭球形状)
       ↓
┌─────────────────────────────┐
│  混沌映射 F (扩散/混淆)      │
│       ↓                      │
│  几何投影 G (约束回椭球面)   │ ← 重复 1000 次
└─────────────────────────────┘
       ↓
密文 = 椭球面上的一个点
```

为什么可靠？

混沌映射 F 几何投影 G
微小差异 → 指数级放大 任意点 → 强制拉回椭球面
抹除轨迹记忆 永不溢出（有界）
正向不可逆 多对一映射（信息耗散）

🔒 安全性摘要

攻击类型 成功概率上界 说明
暴力枚举密钥 2⁻²⁵⁶ 由 SHA-3 哈希派生
已知明文攻击 ≤ 2⁻⁹⁰ 混沌逆向重构不可行
伪造密文 ≤ 2⁻⁹⁰ 需同时命中终点+缠绕数
定时侧信道 0（信息论安全） 恒定时间执行
Grover 加速（量子） 2⁻¹²⁸ 仅平方加速

全域攻破概率：≤ 2⁻⁹⁰（约等于 10⁻²⁷）
