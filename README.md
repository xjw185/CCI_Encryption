# CCI：受约束的混沌迭代加密方法

**Constrained Chaotic Iteration Encryption Method**

> 一种基于混沌动力学与几何约束耦合的对称加密原语。
> 不依赖大数分解或离散对数，具有抗量子计算攻击的潜力。

---

## 📌 项目简介

本项目提出并完整描述了一种**全新的对称加密方法**，其核心思想是：

> 将明文编码为多维空间中的初始状态点，通过**混沌映射**进行扩散与混淆，再通过**几何投影**将状态点约束至紧致曲面（如椭球面）上。两者交替迭代预设轮次后，将最终状态点作为密文输出。

加密密钥由用户口令通过哈希函数动态生成，且生成过程中强制校验混沌轨迹的**拓扑不变量（缠绕数）**与口令匹配，实现“密钥-明文-口令”的三方绑定。

---

## 🧠 核心原理

### 单次迭代结构

每次加密迭代由两步构成：

1. **混沌扩散**
```

(x, y, z) → (x + α·sin(y), y + β·cos(x), z + γ·sin(x+y))  mod 2π

```

2. **几何约束（径向投影至椭球面）**
```

(x, y, z) → (x/r, y/r, z/r)

```
   其中 `r = sqrt((x/a)² + (y/b)² + (z/c)²)`

### 口令缠绕机制

密钥生成时，系统用测试点快速迭代并计算轨迹绕 Z 轴的**缠绕数 W**，强制要求 `W == 用户口令 m`。若不满足，则微调随机数重新生成密钥。

**效果**：口令不再是独立的验证标签，而是直接参与了几何空间的**拓扑结构设计**。

## 📐 系统架构

```

明文数据 + 口令 m
↓
[密钥生成] ← 主密钥 · 时间戳 · 随机数
↓
混沌参数 (α,β,γ) + 几何参数 (a,b,c)
↓
[混合迭代 N 轮]
混沌映射 → 几何投影 → 混沌映射 → 几何投影 → ...
↓
密文 (坐标点 + 随机数)

```

## 🔒 安全性摘要

- **密钥空间**：256 位（由 SHA-3 派生）
- **抗已知明文攻击**：逆向重构成功率 ≤ 2⁻⁹⁰
- **抗伪造攻击**：伪造密文通过验证概率 ≤ 2⁻⁹⁰
- **不依赖**：大数分解、离散对数等量子易解难题
- **安全基础**：混沌不可逆性 + 几何投影的多对一信息耗散
